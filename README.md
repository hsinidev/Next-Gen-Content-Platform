## ğŸ“– Project Description

SEOWriting AI is designed to help creators, marketers, and businesses dominate search rankings. The platform leverages advanced AI to analyze SERP competitors and generate long-form, SEO-optimized content that ranks on Google and gets cited in AI answer engines like ChatGPT and Perplexity.

This repository contains the complete source code for the landing page, featuring a modern, dark-themed UI with an immersive galaxy background, interactive modals, and a built-in SEO guide.

### Key Features
*   **Immersive UI:** Custom animated galaxy background with nebula effects.
*   **SEO-First Architecture:** Pre-rendered HTML with comprehensive JSON-LD Schema (WebSite, Article, FAQPage).
*   **Performance:** Lightweight, zero-dependency structure using Tailwind CSS via CDN.
*   **Content Rich:** Includes an integrated 3500-word SEO guide hidden behind a smooth "Read More" interaction.
*   **Lead Generation:** Optimized CTA placement and "How it Works" funnel.
*   **Compliance:** Fully functional modals for Privacy Policy, Terms of Service, and DMCA.
